003 : Movie Booking Prototype

Worked as a developer in building a prototype of an online movie booking system worth 5000 rupees.The site was made user friendly by extensive use of HTML/CSS ,Bootstrap for the web designing .It consisted of admin and user sections.

The admin section was a password protected interface responsible for adding the movie and the theatres details.This section also granted the admin to design the theatre panel according to the number of seats available in a given theatre.The admin also control adding ,updating deleting and searching the details of movies and theatres.

For faster working of the website AJAX,Jquery and JavaScript were used in the frontend to carry data to the backend.

The user was able to login and signup in the application with access to change his details and password.The user could search his favorite movie and theatre and book the seats available with different time slots.

Once done with booking payment was done using PayPal and the all the booking details could be accessed by the user.All the related pages were built using C#,ASP.NET With MVC framework which provides faster access.

The application was designed using Visual Studio and Microsoft SQL Server.
